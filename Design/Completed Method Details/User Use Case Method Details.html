Class Name UserLoginController
Method Name loginUser
Method State Public
Return Type None
Input Arguments String username, String password
Output Arguments none
Error Messages If the username and password combination isn't valid:
***Invalid username/password.***
If the combination is correct but the User is banned:
***Cannot log you in. You have been banned!***
External Accesses User
External Changes none
Other Methods Invoked authenticate, login, isBanned
Description Obtains username and password data from the HTML form then validates the User's login request
using Django's authenticate function and performs the login with Django's login function.
Banned Users are not allowed to log in.

If the User logs in successfully, they are brought to their profile page.
Otherwise, they are brought back to the login page with the error message.
