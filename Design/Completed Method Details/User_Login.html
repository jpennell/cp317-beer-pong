***Invalid username/password.***

***Cannot log you in. Your account was deactivated.***
Database Accesses User
Database Changes None
Other Methods Invoked authenticate, login, isBanned, isActive
Description Validates the User's login request using Django's authenticate function and performs
the login with Django's login function.
Banned and deactivated Users are not allowed to log in.
