5.3 Software Quality Attributes

5.3.1 Usability

Given the nature of the product's users, the game UI must be as intuitive and user-friendly as possible and cause little frustration.

5.3.2 Interoperability

The website should be usable across varying web browsers in particular the more recent versions of Internet Explorer, Firefox and Google Chrome.

5.3.3 Portability

Users should also be able to access the game UI and website via cell phone.

5.3.4 Maintainability

The client expects to be able to make changes to the website in the future therefore the website must be easy to maintain.

5.3.5 Scalability

The website should have the potential to be expanded to handle users from different regions across the world.

------

Terminology used: website, game UI, product, client, users, regions (nothing special)
